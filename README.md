## O que é CSS3?
De acordo com a Wikipedia :
> CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets (ou simplesmente CSS), onde se define estilos para seu projeto web. Com efeitos de transição, imagem, imagem de fundo/Background e outros, que dão um estilo novo e elegante a seus projetos web. Ou em   todos os aspectos de design do layout da página.
